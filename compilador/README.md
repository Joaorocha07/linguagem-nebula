# Nebula Web Compiler

Compilador/Interpretador online para a linguagem Nebula.

## Como usar

Abra `index.html` em um navegador moderno (Chrome, Firefox, Edge).

## Funcionalidades

- Editor com números de linha
- Análise léxica (Tokens)
- Visualização da AST
- Console de execução
- Documentação integrada
- Exemplos prontos

## Atalhos

- `Ctrl+Enter` - Executar
- `Ctrl+L` - Limpar
